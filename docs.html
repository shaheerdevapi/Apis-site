Email Verification API Documentation

Authentication

All API requests require authentication via an API key sent in the request header:

Authorization: Bearer YOUR_API_KEY

Endpoint

GET /validate

Validates an email address for syntax, domain existence, and basic mailbox checks.

Query Parameters

Parameter Type Required Description
email string Yes Email address to validate

Example Request

GET /validate?email=test@example.com

Sample Response (200 OK)

{
  "email": "test@example.com",
  "valid": true,
  "domain": "example.com"
}

Response Fields

  • email: The email submitted for validation
  • valid: Boolean indicating overall validity
  • domain: Extracted domain from the email
